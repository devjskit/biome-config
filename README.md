# `@devjskit/biome-config`

## Usage

**Install**:

```bash
  yarn add --dev @biomejs/biome @devjskit/biome-config
```

**Edit `package.json`**:

```jsonc
// <project-root>/biome.json
{
  "$schema": "./node_modules/@biomejs/biome/configuration_schema.json",
  "extends": "@devjskit/biome-config",
  // ...
}
```
